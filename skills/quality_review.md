# 质量评审技能

## 评审维度
1. **剧情逻辑** (30分): 矛盾、漏洞、逻辑硬伤
2. **文笔表达** (25分): 描写质量、感官细节、修辞手法
3. **人物一致性** (25分): 性格、对话、动机的连贯性
4. **世界观契合** (20分): 设定违和感

## 评审流程
1. 读取章节全文
2. 逐维度评分
3. 汇总问题清单
4. 提供改进建议

## 常见问题
- AI 套话: "然而""突然""就在这时"等高频模式
- 对话生硬: 所有角色说话风格相似
- 感官缺失: 缺乏视觉、听觉、触觉等细节描写
- 节奏失控: 对话过多或描写过长
