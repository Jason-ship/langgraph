# 章节写作技能

## 适用场景
- 用户需要生成新的小说章节
- 用户需要续写现有章节
- 用户需要对章节进行修改

## 写作标准
1. **字数要求**: 每章 2000-3000 字
2. **结构要求**: 有开头（氛围铺垫）、发展（冲突推进）、结尾（悬念/钩子）
3. **语言要求**: 避免 AI 套话（"然而""突然""就在这时"等高频词）
4. **节奏要求**: 长短句结合，对话与叙述交替

## 输出格式
章节内容直接输出，包含标题但不包含额外说明。

## 质量检查清单
- [ ] 字数达标 (2000+)
- [ ] 有明确的开头/发展/结尾
- [ ] 对话自然，符合角色性格
- [ ] 避免 AI 套话模式
- [ ] 结尾有钩子或悬念
